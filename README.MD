### Prerequisites

Before you begin, ensure you have the following installed:

1. **Clone the repository**

```bash

cd octostudy
```

```bash

npm install

yarn install
```

Open `.env.local` and fill in the required values:

## ğŸ“š API Documentation

### Authentication

All API routes require authentication via Supabase session cookie.

### Endpoints

#### `POST /api/import`
Import new study content.

**Request Body:**
```json
{
  "title": "Biology Chapter 3",
  "sourceType": "text",
  "text": "Content here...",
  "url": null
}
```

**Response:**
```json
{
  "contentId": "uuid",
  "message": "Content imported successfully"
}
```

#### `POST /api/generate`
Generate AI lessons from content.

**Request Body:**
```json
{
  "contentId": "uuid"
}
```

**Response:**
```json
{
  "message": "Lessons generated successfully",
  "lessonIds": ["uuid1", "uuid2", "uuid3"],
  "lessonsCreated": 3,
  "cardsCreated": 9
}
```

#### `GET /api/due`
Fetch cards due for review.

**Response:**
```json
{
  "cards": [...],
  "count": 10
}
```

#### `POST /api/review`
Submit a card review.

**Request Body:**
```json
{
  "cardId": "uuid",
  "correct": true,
  "confidence": "medium"
}
```

**Response:**
```json
{
  "message": "Review recorded successfully",
  "nextDue": "2024-01-15T10:00:00Z",
  "interval": 6
}
```

#### `GET /api/progress`
Get user progress statistics.

**Response:**
```json
{
  "totalCards": 50,
  "dueToday": 10,
  "accuracy": 85,
  "streak": 7,
  "totalReviews": 200,
  "activityData": [...]
}
```
